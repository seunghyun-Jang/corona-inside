# CORONA-INSIDE

### 1. 개요

  코로나 확진자 현황, 백신 접종 현황 정보를 제공해주는 간단한 게시판 기능을 포함한 웹 사이트
  
### 2. 기술 스택
  
  Spring MVC + Hibernate + Spring Security
  
### 3. 제공 기능

  #### 1) 코로나 현황
  
    1-1) URL : https://corona-inside.kr/corona 
    
    1-2) 기능
    
      - 코로나 확진자 수 (어제 확진자 수 / 누적 확진자 수)
      
      - 코로나 확진자 추이 (차트)
      
      - 지역별 코로나 확진자 수 / 거리두기 단계 (전국지도)
      
      - 지역별 코로나 확진자 수 (차트)
